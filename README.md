# Listenwell English Dictation

A browser-only English dictation tool. Upload a `.docx` vocabulary file, extract English words, and read them aloud with an adjustable 3–5 second gap.

## GitHub Pages

In the GitHub repository, open **Settings → Pages**, select **Deploy from a branch**, then select the `main` branch and `/ (root)` folder. Save the setting. GitHub will provide the public site URL.

All file parsing and speech happens locally in the visitor's browser; no Word file is uploaded to a server.
